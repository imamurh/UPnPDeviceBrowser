# UPnPDeviceBrowser

Demo application using [upnpx](https://github.com/fkuehne/upnpx).

![upnpdevicebrowser](https://user-images.githubusercontent.com/18100603/34905602-e6ab1ba8-f89f-11e7-836d-007f7f6b2e37.gif)

- Xcode 9.2
- Swift 4.0.3
- CocoaPods 1.3.1

# Getting the Code
